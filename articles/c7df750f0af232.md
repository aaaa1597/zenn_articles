### 棒グラフ (Bar Chart)
スピリット航空（NK）の月別平均到着遅延時間を示す棒グラフを作成してみましょう。

```python
# 図のサイズを設定
plt.figure(figsize=(10,6))

# タイトルを追加
plt.title("Average Arrival Delay for Spirit Airlines Flights, by Month")

# スピリット航空の月別平均到着遅延を示す棒グラフ
sns.barplot(x=flight_data.index, y=flight_data['NK'])

# 縦軸のラベルを追加
plt.ylabel("Arrival delay (in minutes)")
```
![](https://storage.googleapis.com/zenn-user-upload/416f0df5f40b-20260204.png)

テキスト（タイトルと縦軸ラベル）と図のサイズをカスタマイズするためのコマンドは、前回のチュートリアルで使用したものとお馴染みです。棒グラフを作成するコードは新しく追加しました。
```python
# Bar chart showing average arrival delay for Spirit Airlines flights by month
# スピリット航空便の月別平均到着遅延を示す棒グラフ
sns.barplot(x=flight_data.index, y=flight_data['NK'])
```

棒グラフを作成する sns.barplot には3つの重要な要素があります。
- sns.barplot: 棒グラフを作成することを指示します。
    - sns は seaborn パッケージを指し、このコースでチャートを作成するために使用するすべてのコマンドはこのプレフィックスで始まることに注意してください。
- x=flight_data.index: 横軸に何を使うかを決めます。ここでは、行のインデックス（月）を選択しています。
- y=flight_data['NK']: 棒の高さ（データ）となる列を指定します。この場合は、「NK」列を選択します。

**重要な注意点:** インデックス列を選択するときは必ず flight_data.index を使ってください。flight_data['Month'] と書くとエラーになります。データを読み込む際に "Month" をインデックスとして設定したため、このように特別な書き方をする必要があります。
**インデックス列を選択するには、常にこの特別な表記を使用する必要があります。**

### ヒートマップ (Heatmap)
もう一つのグラフ形式、ヒートマップを学びましょう！ ヒートマップを使うと、データ全体のパターンをひと目で把握できます。各セルは、その数値に応じて色分けされます。

```python
# 図のサイズを設定
plt.figure(figsize=(14,7))

# タイトルを追加
plt.title("Average Arrival Delay for Each Airline, by Month")

# 各航空会社の月別平均遅延を示すヒートマップ
sns.heatmap(data=flight_data, annot=True)

# 横軸のラベルを追加
plt.xlabel("Airline")
```

![](https://storage.googleapis.com/zenn-user-upload/3a7289f8eb17-20260204.png)

ヒートマップ作成のポイントは以下の通りです。
```python
# Heatmap showing average arrival delay for each airline by month
# 各航空会社の月ごとの平均到着遅延を示すヒートマップ
sns.heatmap(data=flight_data, annot=True)
```

このコードには 3 つの主要コンポーネントがあります。
- sns.heatmap: ヒートマップを作成します。
- data=flight_data: データセット内の全数値を使用するよう指示します。
- annot=True: 各セルの中に実際の数値を表示させます（これを False にすると数値が消え、色だけになります）。

この表からどんなパターンが見えますか？例えば、年末に近い月（特に9月〜11月）は、どの航空会社も色が暗（濃）くなっています。これは、この時期は平均してスケジュール通りに運航できていることを示唆しています！
