## 理論編
### はじめに
このチュートリアルでは、Folium パッケージを使ってインタラクティブマップを作成する方法を学びます。その過程で、新しいスキルを応用してボストンの犯罪データを視覚化します。

#### 準備
下記importが必要です。
```python
import pandas as pd
import geopandas as gpd
import math

import folium
from folium import Choropleth, Circle, Marker
from folium.plugins import HeatMap, MarkerCluster
```

### 初めてのインタラクティブマップ
まずは、folium.Map() を使って比較的シンプルなマップを作成します。
```python
# Create a map
m_1 = folium.Map(location=[42.32,-71.0589], tiles='openstreetmap', zoom_start=10)

# Display the map
m_1
```
![](https://storage.googleapis.com/zenn-user-upload/75a0fa04f613-20260107.png)

いくつかの引数によってマップの外観がカスタマイズされます。
**location** は、地図の初期中心位置を設定します。ここではボストン市の緯度（北緯 42.32°）と経度（東経 -71.0589°）を使用しています。

**tiles** は、地図のデザイン（スタイル）を変更します。今回は「OpenStreetMap」スタイルを選択しました。興味があれば、[こちら]から他の選択肢も確認できます。

**zoom_start** は、地図の初期ズームレベルを設定します。数値が大きくなるほど、地図がより拡大（ズームイン）されます。

ここで少し時間をとって、ズームイン・アウトをしたり、地図をドラッグして動かしたりして、自由に触ってみてください。

### ポイントのプロット
地図に収まるデータ量を減らすため、（一時的に）昼間の強盗事件に焦点を絞ります。

daytime_robberies = crimes[((crimes.OFFENSE_CODE_GROUP == 'Robbery') & (crimes.HOUR.isin(range(9,18))))]

### folium.Marker
folium.Marker() を使ってマップにマーカーを追加します。以下の各マーカーはそれぞれ異なる強盗事件に対応しています。
```python
# Create a map
m_2 = folium.Map(location=[42.32,-71.0589], tiles='cartodbpositron', zoom_start=13)

# Add points to the map
for idx, row in daytime_robberies.iterrows():
    Marker([row['Lat'], row['Long']]).add_to(m_2)

# Display the map
m_2
```
![](https://storage.googleapis.com/zenn-user-upload/f1ee5fe757b5-20260107.png)

### folium.plugins.MarkerCluster
追加するマーカーが多数ある場合、folium.plugins.MarkerCluster() を使うとマップを整理するのに役立ちます。各マーカーは MarkerCluster オブジェクトに追加されます。

```python
# Create the map
m_3 = folium.Map(location=[42.32,-71.0589], tiles='cartodbpositron', zoom_start=13)

# Add points to the map
mc = MarkerCluster()
for idx, row in daytime_robberies.iterrows():
    if not math.isnan(row['Long']) and not math.isnan(row['Lat']):
        mc.add_child(Marker([row['Lat'], row['Long']]))
m_3.add_child(mc)

# Display the map
m_3
```
![](https://storage.googleapis.com/zenn-user-upload/03a840dda081-20260107.png)

### バブルマップ
バブルマップでは、マーカーの代わりに円を使用します。円のサイズと色を変えることで、位置と他の2つの変数の関係を示すこともできます。

folium.Circle() を使って円を反復的に追加することで、バブルマップを作成します。下のコードセルでは、9～12時間に発生した強盗は緑色で、13～17時間に発生した強盗は赤色でプロットされています。

```python
# Create a base map
m_4 = folium.Map(location=[42.32,-71.0589], tiles='cartodbpositron', zoom_start=13)

def color_producer(val):
    if val <= 12:
        return 'forestgreen'
    else:
        return 'darkred'

# Add a bubble map to the base map
for i in range(0,len(daytime_robberies)):
    Circle(
        location=[daytime_robberies.iloc[i]['Lat'], daytime_robberies.iloc[i]['Long']],
        radius=20,
        color=color_producer(daytime_robberies.iloc[i]['HOUR'])).add_to(m_4)

# Display the map
m_4
```

![](https://storage.googleapis.com/zenn-user-upload/a88dff4281fd-20260107.png)

folium.Circle() には、いくつかの引数（設定項目）がある点に注目してください。

**location**: 円の中心座標を「緯度・経度」のリスト形式で指定します。

**radius**: 円の半径を設定します。一般的な「バブルマップ」では、各円の半径を変化させることができます。これは、各円の色を変化させるために使った color_producer() 関数と同じような関数を定義することで実装可能です。

**color**: 各円の色を設定します。今回の例では、強盗事件の発生時間帯が場所にどのような影響を与えているかを可視化するために、color_producer() 関数を使用して色分けしています。

### ヒートマップ
ヒートマップを作成するには、folium.plugins.HeatMap() を使用します。これにより、市内の各エリアにおける犯罪の密度を表示できます。赤くなっているエリアほど、相対的に犯罪件数が多いことを示しています。

大都市において予想される通り、ほとんどの犯罪は中心部付近で発生しています。

```python
# Create a base map
m_5 = folium.Map(location=[42.32,-71.0589], tiles='cartodbpositron', zoom_start=12)

# Add a heatmap to the base map
HeatMap(data=crimes[['Lat', 'Long']], radius=10).add_to(m_5)

# Display the map
m_5
```
![](https://storage.googleapis.com/zenn-user-upload/236999e45e0a-20260107.png)
上記のコード セルに示されているように、folium.plugins.HeatMap() はいくつかの引数を取ります。
**data** は、プロットしたい位置を含む DataFrame です。
**radius** はヒートマップの滑らかさを制御します。値が大きいほど、ヒートマップは滑らかになります（つまり、隙間が少なくなります）。

plot_dict が districts と同じインデックスを持つことは非常に重要です。これにより、コードは地理的境界を適切な色と一致させる方法を認識します。
folium.Choropleth() クラスを使用すると、コロプレスマップを作成できます。下の地図が表示されない場合は、別のウェブブラウザでページを表示してみてください。
```python
# Create a base map
m_6 = folium.Map(location=[42.32,-71.0589], tiles='cartodbpositron', zoom_start=12)

# Add a choropleth map to the base map
Choropleth(geo_data=districts.__geo_interface__, 
           data=plot_dict, 
           key_on="feature.id", 
           fill_color='YlGnBu', 
           legend_name='Major criminal incidents (Jan-Aug 2018)'
          ).add_to(m_6)

# Display the map
m_6
```
![](https://storage.googleapis.com/zenn-user-upload/9c39efc28032-20260107.png)

folium.Choropleth() には、いくつかの引数がある点に注目してください。

**geo_data**: 各地理的エリアの境界線データを含む「GeoJSON FeatureCollection」を指定します。上記のコードでは、districts という GeoDataFrame を __geo_interface__ 属性を使って GeoJSON 形式に変換しています。

**data**: 各エリアを塗り分ける際の数値（基準となる値）が入った「Pandas Series」を指定します。

**key_on**: 常に feature.id に設定します。これは、geo_data に渡したデータと、data に渡した Pandas Series が同じインデックス（ID）を持っていることを紐付けるためのものです。

詳しく解説すると、GeoJSON の構造（"features" キーに対応する値がリストになっており、各項目が "id" キーを持つ辞書形式であること）に基づいた指定方法です。

**fill_color**: 塗りつぶしの色のスケール（カラーパレット）を設定します。

**legend_name**: 地図の右上に表示される凡例（レジェンド）にラベルを付けます。
