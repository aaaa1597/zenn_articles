# Androidアプリの流用開発
今どき、githubみれば、サンプルアプリがゴロゴロ転がってて、いい時代だな～と思う、今日この頃。皆様におかれましては、いかがお過ごしでしょうか？
githubからcloneして動かすことが出来ると、次は自分のアプリを作りたくなるに違いない。
作るにしても、流用開発が楽でいいよね。
その時の手順をまとめてみた。

# 流用開発の手順
1.AndroidManifest.xmlを開く→package名の変更
　※リファクタリング→名前の変更、から変更すると幸せになれます。
　※↑だけど.区切りに増減が増えると、できなくなるので、そこは力技で。

2.モジュールのbuild.gradleのapplicationIdをpackage名と同じ名前に変更
　※Sync Nowってのが出るならそのボタンを押下、でなければ、AndroidStudio再起動。

3.JNI関数の関数名を変更。
　※ここは状況に応じてなので、気合で。

以上。
これで、できるはず。
