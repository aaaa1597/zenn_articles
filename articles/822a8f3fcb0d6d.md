### 散布図
シンプルな散布図を作成するには、sns.scatterplot コマンドを使用し、以下の値を指定します。
- 水平X軸（x=insurance_data['bmi']）
- 垂直Y軸（y=insurance_data['charges']）

```python
sns.scatterplot(x=insurance_data['bmi'], y=insurance_data['charges'])
```
![](https://storage.googleapis.com/zenn-user-upload/954e1934d40d-20260204.png)

上記の散布図は、ボディマス指数（BMI）と保険料の間に「正の相関」があることを示唆しています。つまり、BMIが高い顧客ほど、保険料も高くなる傾向にあります。（高いBMIは一般的に慢性疾患のリスク上昇に関連しているため、この傾向は理にかなっていると言えます。）

この関係性の強さを再確認するために、回帰直線（データに最も適合する直線）を追加してみるのもよいでしょう。これを行うには、コマンドを sns.regplot に変更するだけで済みます。

```python
sns.regplot(x=insurance_data['bmi'], y=insurance_data['charges'])
```
![](https://storage.googleapis.com/zenn-user-upload/54917a10f1e7-20260204.png)

### 色分けされた散布図
散布図を使えば、（2つだけでなく……）3つの変数間の関係を表示することができます！その方法の一つが、点の色分けです。

例えば、喫煙がBMIと保険料の関係にどのような影響を与えるかを理解するために、点を「喫煙の有無（smoker）」で色分けし、残りの2つの項目（「bmi」と「charges」）を軸に配置してプロットすることができます。

```python
sns.scatterplot(x=insurance_data['bmi'], y=insurance_data['charges'], hue=insurance_data['smoker'])
```

![](https://storage.googleapis.com/zenn-user-upload/c4a9b02c4960-20260204.png)

この散布図は、非喫煙者は BMI の上昇とともにわずかに多く支払う傾向があるのに対し、喫煙者ははるかに多く支払う傾向があることを示しています。

```python
sns.lmplot(x="bmi", y="charges", hue="smoker", data=insurance_data)
```

![](https://storage.googleapis.com/zenn-user-upload/737e5e12f31e-20260204.png)

上記の sns.lmplot コマンドは、これまで学んできたコマンドとは少し使い勝手が異なります。

これまでは、'bmi' 列を選択するために x=insurance_data['bmi'] と記述していましたが、このコマンドでは x="bmi" と、列名のみを指定します。同様に、y="charges" や hue="smoker" にも列名を入力します。そして、data=insurance_data という形でデータセット全体を指定します。

最後に、もう一つ別のグラフを紹介します。これは、皆さんが見慣れている散布図とは少し見た目が違うかもしれません。通常、散布図は（"bmi" と "charges" のような）2つの連続変数の関係を強調するために使われます。しかし、散布図のデザインを応用して、主軸の片方に（"smoker" のような）カテゴリ変数を配置することもできます。このようなグラフを「カテゴリ散布図（categorical scatter plot）」と呼び、sns.swarmplot コマンドで作成します。

```python
sns.swarmplot(x=insurance_data['smoker'], y=insurance_data['charges'])
```

![](https://storage.googleapis.com/zenn-user-upload/45d9db23c3f5-20260204.png)

このグラフからわかることは以下の通りです。
- 平均すると、非喫煙者は喫煙者よりも料金が安い。
- 最も高い料金を支払っている顧客は喫煙者であり、最も低い料金を支払っている顧客は非喫煙者です。
