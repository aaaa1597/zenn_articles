# Ubuntuに go をインストール
[Download and install](https://go.dev/doc/install)を参考に。
```shell
$ cd ~ && mkdir tempgo && cd tempgo
$ wget https://go.dev/dl/go1.21.4.linux-amd64.tar.gz
$ sudo rm -rf /usr/local/go
$ sudo tar -C /usr/local -xzf go1.21.4.linux-amd64.tar.gz
$ rm -rf ~/tempgo
$ echo 'export PATH="$PATH:/usr/local/go/bin"' >> ~/.bash_profile
$ source ~/.bash_profile
$ go version
```

# go のプロジェクト生成して動かしてみる
goプロジェクト初期化のおまじない
```shell
$ cd ~ && mkdir hellogo && cd hellogo
$ go mod init hellogo
```

goのサンプルコードを生成
```shell
$ vi helloworldgo.go
```
```go
package main

import "fmt"

func main() {
  fmt.Printf("Hello World\n")
}
```
go実行
```shell
$ go run helloworldgo.go
```
できた!!

build -> go実行
```shell
$ go build helloworldgo.go
```

実行
```shell
$ ./helloworldgo
```
できた!!

# GoでWebアプリを動かす
```shell:webアプリ作成
$ cd ~ && mkdir test-go-webapp && cd test-go-webapp
$ go mod init test-go-webapp
$ vi webmain.go
```

webmain.goの内容はこれ。
```go:webmain.go
package main
import (
    "fmt"
    "net/http"
)
func main() {
    http.HandleFunc("/", HelloServer)
    http.ListenAndServe(":8080", nil)
}
func HelloServer(w http.ResponseWriter, r *http.Request) {
    fmt.Fprintf(w, "Hello, %s!", r.URL.Path[1:])
}
```

# Webアプリ実行
```shell:webアプリ実行
$ go run webmain.go
```

ブラウザーを開いて、http://localhost:8080/world にアクセスする。
![](https://storage.googleapis.com/zenn-user-upload/b76efd4c6241-20231202.png)
出来た!! Hello, world!が表示された。

終了するときは、terminalで、"ctrl+c"ね。
