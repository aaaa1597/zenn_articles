# 手順
## 1.プロジェクト生成 -> VSCodeで開く
めんどいから、プロジェクト雛形をgithubから持ってくる。[React+TypeScriptのプロジェクト雛形をgithubに置いとく話。](https://zenn.dev/rg687076/articles/cb8531be2d5cd6)

## 2.生成したプロジェクトの場所でreact-zxingをインストール
```shell:
$ cd プロジェクトのフォルダ
$ npm install
$ npm install react-zxing --save
$ npm install styled-components --save
```
