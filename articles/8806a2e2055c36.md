# Abstract
Webアプリ開発の勉強を初めたけど、無料で公開できるサイトがない!!
と彷徨ってたら、Githubの"Github Pages"というナイスなサービスがあるのを知った。
その方法を残しときます。

# 前提
[React+TypeScriptのプロジェクトをGithubに登録してみる。](https://zenn.dev/rg687076/articles/fc4f3c627820fd)のコードを公開する。

:::message alert
useLoader使っている時は、相対パス指定にせんと、ローカルで動いてても、pages上げたら動かんくなる。
気をつけて!!
```
const fbx = useLoader(FBXLoader, "assets/Ch09_nonPBR.fbx"); ### ← 相対パス
```
:::

:::message alert
useFBX使っている時は、'./'始まりの相対パスにせんと、動かんくなる。
気をつけて!!
```
anim = useFBX('./assets/BreakdanceEnding2.fbx').animations ### ← "./"始まりの相対パス
```
:::
