# 手順
## git clone ～ npm startまで
```shell:git clone
$ cd ~
## 1.githubからgit clone
$ git clone https://github.com/aaaa1597/react-helloworld.git
## 2.ディレクトリ名を適当に変更
$ mv react-helloworld aaa && cd aaa
## 3.npm install
$ npm install
## 4.npm start
$ npm start
```
↓
出来た!!

ブラウザ閉じて、Terminalにctrl+cで終了。
意外と簡単だった。
