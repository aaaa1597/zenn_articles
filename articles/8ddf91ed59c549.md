# 手順
## 1.プロジェクト生成 -> VSCodeで開く
めんどいから、プロジェクト雛形をgithubから持ってくる。[React+TypeScriptのプロジェクト雛形をgithubに置いとく話。](https://zenn.dev/rg687076/articles/cb8531be2d5cd6)

## 2.生成したプロジェクトの場所でreact-zxingをインストール
```shell:
$ cd プロジェクトのフォルダ
$ npm install
$ npm install react-zxing --save
```

## 3.App.tsxを修正
```diff ts:App.tsx
import React from 'react';
import './App.css';
+import { useState } from "react";
+import { useZxing } from "react-zxing";

function App() {
+ const [result, setResult] = useState("");
+ const { ref } = useZxing({
+   onDecodeResult(result) {
+     setResult(result.getText());
+   },
+ });
  
  return (
    <div className="App">
-      hello world!!
+     <video ref={ref} />
+     <p>
+       <span>Last result:</span>
+       <span>{result}</span>
+     </p>
    </div>
  );
}

export default App;
```
