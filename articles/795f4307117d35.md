# Abstract
VirtualBoxに日本語版Ubuntuをインストールする話。
日本語環境じゃないと、キーボードがUSキーボードで、割り当てが変だったりして、意外にめんどい。だったら、最初から日本語版をインストールすればいいじゃねってことで。

# 前提
VirtualBoxはインストールしておくこと。
[ココ](https://qiita.com/HirMtsd/items/be1c8afe708e901e1100)が参考になる。

# Ununtuイメージをダウンロード
とはいえ、日本語版ubuntuイメージをゲットできれば、あとの手順はさして迷うことなく。
[↓ここ](https://www.ubuntulinux.jp/News/ubuntu2204-ja-remix)から、(赤丸のリンク先はどこでもよくて)、"ubuntu-ja-22.04-desktop-amd64.iso"をダウンロードする。

# 必要情報を入力
## 名前とオペレーションシステム
![](https://storage.googleapis.com/zenn-user-upload/4864ecf397b8-20231125.png)
名前 : 適当で。
フォルダー : 適当でいいけど、容量は結構消費するので、十分空きのあるドライブを選ぶ。
ISOイメージ : ここ大事。上記でダウンロードしたisoファイルを選ぶ。
タイプ : Linuxで。
バージョン : Ubuntu(64-bit)で。

## 自動インストール
![](https://storage.googleapis.com/zenn-user-upload/f3e68f4fa12d-20231125.png)
ユーザー: 適当で。
パスワード : 適当で。
パスワードの再入力 : 適当で。パスワードと同じに。
ホスト名 : 適当で。
ドメイン名 : 適当で。
Guest Additionsのチェックはどっちでも。

## ハードウェア
![](https://storage.googleapis.com/zenn-user-upload/5e3cbcd5711a-20231125.png)
メインメモリー : 適当で。多めがいい。
プロセッサ数 : 適当で。多めがいい。
EFIを有効化のレ点は外す。これにチェックすると起動しなかった。

## ハードディスク
![](https://storage.googleapis.com/zenn-user-upload/a4e1b0df0c13-20231125.png)
仮想ハードディスクを作成するにチェック。
ハードディスクファイルの場所とサイズ: 適当で。容量に気を付けて。
ハードディスクのファイルタイプと種類 : 適当で。デフォルトのVDIでいい。全サイズの事前割り当てはチェックしないでおく。

# ホストOS(Windwos)⇔ゲストOS(Ununtu)間のコピペ設定
このままだと、ホストOS(Windwos)⇔ゲストOS(Ununtu)間でコピペができない。
なので、Guest Additionsをインストールする。
![](https://storage.googleapis.com/zenn-user-upload/d1005ecd201b-20231126.png)
↓
Guest Additionsがマウントされたのを確認 -> ダブルクリック
![](https://storage.googleapis.com/zenn-user-upload/69482054b519-20231126.png)
↓
VBoxLinuxAdditions.runを右クリックして、プロパティを確認する。
![](https://storage.googleapis.com/zenn-user-upload/ff5103b62d45-20231126.png)
↓
![](https://storage.googleapis.com/zenn-user-upload/3dada7c92fe6-20231126.png)
↓
VBoxLinuxAdditions.runのパスを確認
![](https://storage.googleapis.com/zenn-user-upload/1bbf7d18937f-20231126.png)
↓
Terminal起動
![](https://storage.googleapis.com/zenn-user-upload/55ee06e8065a-20231126.png)
↓
bashをsudoで実行すため、sudoリストに自分ユーザを追加する
```shell:sudo追加
$ su
$ gpasswd -a $USER sudo
$ getent group sudo
sudo:x:27:aaaa ### <- aaaaが追加された
```
再起動
↓
さっきの確認したパスに移動 -> VBoxLinuxAdditions.runを実行
```shell:VBoxLinuxAdditions.run実行
cd /media/aaaa/VBox_GAs_7.0.12
sudo bash ./VBoxLinuxAdditions.run
```
↓
Guest Additionsは、もう不要なのでアンマウント->取り出ししとく。
![](https://storage.googleapis.com/zenn-user-upload/43274a01fd42-20231126.png)
↓
デバイス -> クリップボードの共有 -> 双方向
![](https://storage.googleapis.com/zenn-user-upload/f873f473801a-20231126.png)
↓
デバイス -> ドラッグ＆ドロップ -> 双方向
![](https://storage.googleapis.com/zenn-user-upload/22c0717a3e7c-20231126.png)
コピペの設定完了
再起動
