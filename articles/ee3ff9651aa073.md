# グループ化と並べ替え
洞察力のレベルを高めます。データセットが複雑になるほど、これはより重要になります。
## 理論編

### はじめに
マップを使用すると、DataFrame または Series 内のデータを、列全体にわたって一度に 1 つの値ずつ変換できます。しかし、多くの場合、データをグループ化し、そのグループに固有の処理を実行したい場合があります。

これから学ぶように、これは groupby() 操作を使って行います。また、DataFrame のインデックスを作成するより複雑な方法や、データの並べ替え方法など、いくつかの追加トピックについても説明します。

**準備**
```python
import pandas as pd
reviews = pd.read_csv("../input/wine-reviews/winemag-data-130k-v2.csv", index_col=0)
pd.set_option("display.max_rows", 5)
```

#### グループ化分析
これまで頻繁に使用してきた関数の一つに value_counts() があります。value_counts() の機能と同じことを、次のように記述することで再現できます。
```python
reviews.groupby('points').points.count()
```
**結果**
```
points
80     397
81     692
      ... 
99      33
100     19
Name: points, Length: 21, dtype: int64
```

groupby() は、指定されたワインに同じポイント値を割り当てたレビューのグループを作成しました。次に、各グループについて points() 列を取得し、その出現回数をカウントしました。value_counts() は、この groupby() 操作への単なるショートカットです。

このデータには、これまで使用してきた集計関数をすべて使用できます。例えば、各ポイント値カテゴリで最も安いワインを取得するには、次のようにします。
```python
reviews.groupby('points').price.min()
```
**結果**
```
points
80      5.0
81      5.0
       ... 
99     44.0
100    80.0
Name: price, Length: 21, dtype: float64
```

生成する各グループは、一致する値を持つデータのみを含む DataFrame のスライスと考えることができます。この DataFrame には apply() メソッドを使用して直接アクセスでき、必要に応じてデータを操作できます。
たとえば、データセット内の各ワイナリーから最初にレビューされたワインの名前を選択する方法の 1 つを次に示します。
```python
reviews.groupby('winery').apply(lambda df: df.title.iloc[0])
```


```python
reviews.groupby(['country', 'province']).apply(lambda df: df.loc[df.points.idxmax()])
```

もう一つの注目すべきgroupby()メソッドはagg()です。これを使うと、DataFrameに対して複数の異なる関数を同時に実行できます。例えば、以下のようにデータセットの簡単な統計サマリーを生成できます。
```python
reviews.groupby(['country']).price.agg([len, min, max])
```

groupby() を効果的に使用すると、データセットで非常に強力な機能を多数実行できるようになります。

#### マルチインデックス
これまで見てきたすべての例では、単一ラベルのインデックスを持つ DataFrame オブジェクトまたは Series オブジェクトを扱ってきました。
groupby() は、実行する操作に応じて、マルチインデックスと呼ばれる結果になることがあるという点で若干異なります。
マルチインデックスは、複数のレベルを持つという点で通常のインデックスとは異なります。
例:
```python
countries_reviewed = reviews.groupby(['country', 'province']).description.agg([len])
countries_reviewed
```
**結果**
|-<br/>country|-<br/>province|len<br/>-|
|---|---|---|
|Argentina|Mendoza Province|3264|
|Other|536|
|...|...|...|
|Uruguay|San Jose|3|
|Uruguay|24|

425 rows × 1 columns

```python
mi = countries_reviewed.index
type(mi)
```
**結果**
```
pandas.core.indexes.multi.MultiIndex
```
マルチインデックスには、単一レベルインデックスにはない、階層構造を処理するための方法がいくつかあります。また、値を取得するには2階層のラベルが必要です。複数インデックスの出力を扱うことは、Pandasを初めて使うユーザーにとってよくある「落とし穴」です。

マルチインデックスの使用例と使用手順については、Pandas ドキュメントの[「MultiIndex / Advanced Selection」](https://pandas.pydata.org/pandas-docs/stable/user_guide/advanced.html)セクションで詳しく説明されています。

とはいえ、一般的に最も頻繁に使用されるマルチインデックス メソッドは、通常のインデックスに戻す reset_index() メソッドです。これを覚えておきましょう。
```python
countries_reviewed.reset_index()
```
**結果**

|---|---|---|---|
|0|Argentina|Mendoza Province|3264|
|1|Argentina|Other|536|
|...|...|...|...|
|423|Uruguay|San Jose|3|
|424|Uruguay|Uruguay|24|

425 rows × 3 columns

#### ソート
countries_reviewed をもう一度見てみると、グループ化によって値が値順ではなくインデックス順で返されることがわかります。つまり、groupby の結果を出力する場合、行の順序はデータではなくインデックスの値に依存します。

データを必要な順序に並べ替えるには、自分で並べ替える必要があります。sort_values() メソッドは便利です。
```python
countries_reviewed = countries_reviewed.reset_index()
countries_reviewed.sort_values(by='len')
```
**結果**
||country|province|len|
|---|---|---|---|
|179|Greece|Muscat of Kefallonian|1|
|192|Greece|Sterea Ellada|1|
|...|...|...|...|
|415|US|Washington|8639|
|392|US|California|36247|

425 rows × 3 columns

sort_values() はデフォルトで昇順ソート（最も小さい値が先頭）になります。しかし、ほとんどの場合、降順ソート（大きい値が先頭）にしたいでしょう。つまり、以下のようになります。
```python
countries_reviewed.sort_values(by='len', ascending=False)
```
**結果**
||country|province|len|
|---|---|---|---|
|392|US|California|36247|
|415|US|Washington|8639|
|...|...|...|...|
|63|Chile|Coelemu|1|
|149|Greece|Beotia|1|

425 rows × 3 columns

インデックス値でソートするには、コンパニオンメソッドのsort_index()を使用します。このメソッドの引数とデフォルトの順序は、以下の通りです。
```python
countries_reviewed.sort_index()
```
**結果**
||country|province|len|
|---|---|---|---|
|0|Argentina|Mendoza Province|3264|
|1|Argentina|Other|536|
|...|...|...|...|
|423|Uruguay|San Jose|3|
|424|Uruguay|Uruguay|24|

425 rows × 3 columns

最後に、一度に複数の列で並べ替えることができることに注意してください。
```python
countries_reviewed.sort_values(by=['country', 'len'])
```
||country|province|len|
|---|---|---|---|
