"Stable Diffusion"使いに捧ぐ。

# Abstract
AdobeのPhotoshopとかだと設定できるらしいけど、 Adobe税が高いから。
「Stable Diffusion」だと無料で出来るのが分かったので、載せておきます。
※ポイントは「HakuImg」の使い方。

# 前提
「Stable Diffusion」はインストールしておく。
参照: [[無料][環境構築][生成AI]Stable Diffusionを動かしてみた。2024.02(ローカルPC)](https://zenn.dev/rg687076/articles/b77ee55bda37dc)

# 前準備「HakuImg」をインストールする。

1. 「Stable Diffusion」起動
2. 「HakuImg」インストール
   Extensions → Available → "Load from:"を押下
   ズラーと拡張機能の一覧が表示される。
![](https://storage.googleapis.com/zenn-user-upload/a0648fc5ce83-20240312.png)
3. 検索BOXに「haku」と入力 → 絞る → "Install"押下
![](https://storage.googleapis.com/zenn-user-upload/f79e93041482-20240312.png)
4. インストールされているのを確認する。
![](https://storage.googleapis.com/zenn-user-upload/1b359a40764a-20240312.png)
5.Settings → "Reload UI"を押下して、「Stable Diffusion」を再起動。
![](https://storage.googleapis.com/zenn-user-upload/9bfa046d751c-20240312.png)
インストール完了!!

# 手順
1. 「Stable Diffusion」起動
2. 「HakuImg」→ Effect → 色域変更する画像をドラッグ＆ドロップする。
![](https://storage.googleapis.com/zenn-user-upload/317a571bddb5-20240312.png)
3. 「HakuImg」→ Effect → 色域変更する画像をドラッグ＆ドロップする。
![](https://storage.googleapis.com/zenn-user-upload/515f1bfa468b-20240312.png)
4. hueのつまみを変更すると色域をいい感じに変更してくれる。
![](https://storage.googleapis.com/zenn-user-upload/8af3fe05362c-20240312.png)
5. 後は、変換後の画像を右クリック → 名前を付けて保存で保存すれば完了。

完成!! 便利～!!
