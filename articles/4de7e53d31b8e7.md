## 座標参照系(Coordinate Reference Systems(CRS))
地球の表面を2次元（平面）で表現できるなんて、かなり驚くべきことですよね！

## 理論編
### 0.はじめに
このコースで作成する地図は、地球の表面を2次元（平面）で表現したものです。しかし、ご存知の通り、現実の地球は3次元の球体（地球儀）です。そのため、地球を平らな面として描画するには、**地図投影法（Map Projection）** と呼ばれる手法を使う必要があります。

地図投影法は、100%正確というわけにはいきません。どの投影法も、何らかの有用な特性を保持する一方で、地球の表面のどこかに必ず歪みを生じさせます。例えば、以下のようなものがあります。

**正積図法（Equal-area projections）:** （例：「ランベルト円筒正積図法」や「アフリカ・アルベルス正積円錐図法」） 面積を正しく保持します。国や都市の面積を計算したい場合に適した選択です。

**正距図法（Equidistant projections）:** （例：「正距方位図法」） 距離を正しく保持します。飛行機の飛行距離を計算する場合などに適しています。

![](https://storage.googleapis.com/kaggle-media/learn/images/noBRRNR.png)

私たちは、投影された点と地球上の実際の場所がどのように対応するかを示すために、**座標参照系（CRS: Coordinate Reference System）** を使用します。このチュートリアルでは、座標参照系の詳細と、GeoPandas（Pythonのライブラリ）での使用方法について学んでいきます。

# Create a map

### 幾何学的オブジェクトの属性
最初のチュートリアルで学んだように、GeoDataFrameの「geometry（幾何情報）」列のデータ型は、表現したいものによって異なります。例えば：
- Point（点）: 地震の震源地など
- LineString（線）: 通り（道路）など
- Polygon（ポリゴン/多角形）: 国の境界線など

これら3つの幾何学的オブジェクトには、データセットを素早く分析するために活用できる**属性(Attributes)** が組み込まれています。例えば、x属性と y属性を使えば、Point（点）のx座標とy座標をそれぞれ取得できます。
```python: 例:
# 各ポイントのx座標を取得
facilities.geometry.head().x
```
また、LineStringからは length(長さ) 属性を、Polygonからは area(面積) 属性を取得することができます。

```python
# 各ポリゴンの面積（平方メートル）を計算
# 10の6乗で割って平方キロメートルに変換
regions.loc[:, "AREA"] = regions.geometry.area / 10**6

print("ガーナの面積: {} 平方キロメートル".format(regions.AREA.sum()))
print("CRS:", regions.crs)
regions.head()
```
