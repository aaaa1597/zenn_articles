## 理論編
### 時系列(Time Series)へようこそ！
**予測（forecasting）** は、現実世界における機械学習の応用として、おそらく最も一般的なものです。企業は製品需要を予測し、政府は経済成長や人口増加を予測し、気象学者は天気を予測します。将来を理解することは、科学・政府・産業（さらには私たちの個人生活においても！）に共通する切実な課題であり、これらの分野の実務家たちはこの課題に取り組むため、ますます機械学習を活用するようになっています。

時系列予測は、長い歴史を持つ広範な分野です。本コースでは、時系列データに対して現代的な機械学習手法を適用し、できる限り正確な予測を行うことを目標とします。本コースのレッスンは、過去のKaggle予測コンペティションの優勝解法に着想を得ていますが、正確な予測が重視されるあらゆる場面で応用可能です。

このコースを修了すると、次のことができるようになります：

* 時系列の主要な構成要素（トレンド、季節性、循環）をモデル化するための特徴量エンジニアリング
* さまざまな種類の時系列プロットによる可視化
* 相補的なモデルの強みを組み合わせたハイブリッド予測モデルの作成
* さまざまな予測タスクに機械学習手法を適応させる方法の習得

演習の一環として、「[Store Sales - Time Series Forecasting](https://www.kaggle.com/c/store-sales-time-series-forecasting) Getting Started」コンペティションにも参加できます。このコンペでは、エクアドルに拠点を置く大手食料品小売業者 Corporación Favorita の約1800の商品カテゴリについて売上を予測する課題に取り組みます。

### 時系列とは何か？

予測の基本対象は「時系列（time series）」です。これは、時間の経過とともに記録された観測値の集合を指します。予測の応用では、観測値は通常、日次や月次など一定の頻度で記録されます。

```python
import pandas as pd

df = pd.read_csv(

    index_col='Date',
    parse_dates=['Date'],
).drop('Paperback', axis=1)

df.head()
```

### 時系列における線形回帰

このコースの前半では、予測モデルの構築に線形回帰アルゴリズムを使用します。線形回帰は実務で広く用いられており、複雑な予測タスクにも自然に適応できます。

**線形回帰**アルゴリズムは、入力特徴量の重み付き和を学習します。2つの特徴量がある場合、次のようになります：

`target = weight_1 × feature_1 + weight_2 × feature_2 + bias`

学習の過程で、回帰アルゴリズムは target に最もよく適合するように、weight_1、weight_2、bias の値を学習します。（このアルゴリズムは通常、予測値と実測値の二乗誤差を最小化することから「最小二乗法」と呼ばれます。）weight は回帰係数、bias は切片とも呼ばれます。切片は、この関数のグラフが y 軸と交わる位置を示します。

### 時間ステップ特徴量
時系列特有の特徴量には2種類あります：時間ステップ特徴量とラグ特徴量です。

時間ステップ特徴量は、時間インデックスから直接作成できる特徴量です。最も基本的なものは「タイムダミー（time dummy）」で、系列の最初から最後まで時間ステップを順に数えたものです。

```python
import numpy as np

df['Time'] = np.arange(len(df.index))

df.head()
```

| Date       | Hardcover | Time |
| ---------- | --------- | ---- |
| 2000-04-01 | 139       | 0    |
| 2000-04-02 | 128       | 1    |
| 2000-04-03 | 172       | 2    |
| 2000-04-04 | 139       | 3    |
| 2000-04-05 | 191       | 4    |

タイムダミーを使った線形回帰モデルは次のようになります：

`target = weight × time + bias`

Time を x 軸に取ることで、時系列プロット上に曲線（トレンド）を当てはめることができます。


時間ステップ特徴量を使うことで **時間依存性（time dependence）** をモデル化できます。
ある系列が時間依存であるとは、その値が「いつ発生したか」から予測できることを意味します。Hardcover 売上の系列では、月の後半の売上は前半よりも一般的に高いと予測できます。

### ラグ特徴量

ラグ特徴量を作るには、目的系列の観測値を時間的にずらします。ここでは1ステップのラグを作成していますが、複数ステップずらすことも可能です。

```python
df['Lag_1'] = df['Hardcover'].shift(1)
df = df.reindex(columns=['Hardcover', 'Lag_1'])

df.head()
```

| Date       | Hardcover | Lag_1 |
| ---------- | --------- | ----- |
| 2000-04-01 | 139       | NaN   |
| 2000-04-02 | 128       | 139   |
| 2000-04-03 | 172       | 128   |
| 2000-04-04 | 139       | 172   |
| 2000-04-05 | 191       | 139   |

ラグ特徴量を用いた線形回帰モデルは次のようになります：

`target = weight × lag + bias`

ラグ特徴量を使うことで、現在の値を前日の値に対してプロットする「ラグプロット」に曲線を当てはめることができます。

```python
fig, ax = plt.subplots()
ax = sns.regplot(x='Lag_1', y='Hardcover', data=df, ci=None, scatter_kws=dict(color='0.25'))
ax.set_aspect('equal')
ax.set_title('Lag Plot of Hardcover Sales');
```

![](https://storage.googleapis.com/zenn-user-upload/f2b68926addb-20260226.png)

ラグプロットを見ると、ある日の売上（Hardcover）が前日の売上（Lag_1）と相関していることがわかります。このような関係が見られる場合、ラグ特徴量が有効であることがわかります。

より一般的に言えば、ラグ特徴量は**系列依存（serial dependence）** をモデル化します。
時系列が系列依存を持つとは、ある観測値が過去の観測値から予測できることを意味します。Hardcover 売上では、ある日の売上が高ければ翌日も高い傾向があると予測できます。

直線の傾きと切片の式を覚えていますか？傾きは3.33なので、このモデルによれば、ハードカバーは時間1ステップごとに平均3.33単位変化します。

回帰係数を解釈することで、時系列プロットにおける系列依存（serial dependence）を認識することができます。`target = weight × lag_1 + error`というモデルを考えてみましょう。ここで error はランダムノイズ、weight は -1 から 1 の間の値です。この場合、weight は「次の時点の値が、前の時点と同じ符号をとる可能性がどれくらい高いか」を示しています。

* weight が 1 に近い → 次の値は前の値と同じ符号になる可能性が高い
* weight が -1 に近い → 次の値は前の値と反対の符号になる可能性が高い
