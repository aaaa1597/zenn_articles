## 理論編
### プロットタイプとカスタムスタイルの選択
チャートをカスタマイズして、おしゃれに見せましょう。

このコースでは、さまざまな種類のグラフを作成する方法を学びました。次に、これまでの知識を整理し、グラフのスタイルを変更するための簡単なコマンドをいくつか学びます。

学んだことの整理
![](https://storage.googleapis.com/kaggle-media/learn/images/LPWH19I.png)

データが持つ背景（ストーリー）を伝えるのに、どのグラフが最適か判断するのは必ずしも簡単ではありません。そのため、判断の助けとなるよう、チャートのタイプを大きく3つのカテゴリーに分類しました。

1. トレンド (Trends)
トレンドとは「変化のパターン」のことです。
    - sns.lineplot（折れ線グラフ）: 一定期間の変化（トレンド）を示すのに最適です。複数の線を用いることで、複数のグループのトレンドを比較できます。

2. 関係性 (Relationship)
データ内の変数間の関係を理解するために、多くのチャートタイプが利用できます。
- sns.barplot（棒グラフ）: 異なるグループ間の数量を比較するのに便利です。
- sns.heatmap（ヒートマップ）: 数値表の中にあるパターンを、色分けして見つけ出すのに使われます。
- sns.scatterplot（散布図）: 2つの連続変数間の関係を示します。色分け（color-coded）をすれば、3つ目のカテゴリ変数との関係も表現できます。
- sns.regplot（回帰グラフ）: 散布図に回帰線を加えることで、2変数間の線形関係（直線的な関係）が見やすくなります。
- sns.lmplot: 散布図に色分けされた複数のグループが含まれる場合、複数の回帰線を描くのに便利です。
- sns.swarmplot（群群プロット/スウォームプロット）: カテゴリ別散布図。連続変数とカテゴリ変数の関係を示します。

3. 分布 (Distribution)
分布を視覚化することで、変数にどのような値が想定されるか、またそれらがどの程度の確率で発生するかを示します。
- sns.histplot（ヒートグラム）: 単一の数値変数の分布を示します。
- sns.kdeplot（カーネル密度推定プロット）: 単一（または2つ）の数値変数の分布を、滑らかに推定して表示します。
- sns.jointplot: 2D KDEプロットと、それぞれの変数に対応するKDEプロットを同時に表示するのに役立ちます。

### Seabornでのスタイル変更
これまでのコマンドはすべて、デフォルトで綺麗なスタイルが適用されていました。しかし、グラフの外観をカスタマイズしたい場合もあるでしょう。幸い、これはコードをたった1行追加するだけで実現できます。

いつものように、まずはコーディング環境のセットアップから始めます（以下のコードは非表示ですが、右下の「Code」ボタンをクリックすると表示されます）。

```python
import pandas as pd
pd.plotting.register_matplotlib_converters()
import matplotlib.pyplot as plt
%matplotlib inline
import seaborn as sns
print("Setup Complete")
```
Setup Complete

以前のチュートリアルで使用した折れ線グラフのコードを使います。以下のコードはデータセットを読み込み、チャートを作成します。

```Python
# 読み込むファイルのパス
spotify_filepath = "../input/spotify.csv"

# ファイルを読み込み、変数 spotify_data に格納
spotify_data = pd.read_csv(spotify_filepath, index_col="Date", parse_dates=True)

# 折れ線グラフ
plt.figure(figsize=(12,6))
sns.lineplot(data=spotify_data)
```
![](https://storage.googleapis.com/zenn-user-upload/732725896e34-20260204.png)
たった1行のコードで、図のスタイルを別のテーマに素早く変更できます。

```python
# 図のスタイルを "dark" テーマに変更
sns.set_style("dark")

# 折れ線グラフ
plt.figure(figsize=(12,6))
sns.lineplot(data=spotify_data)
```
![](https://storage.googleapis.com/zenn-user-upload/3f66932002d6-20260204.png)

Seabornには5つのテーマがあります：(1) "darkgrid", (2) "whitegrid", (3) "dark", (4) "white", (5) "ticks"。 上記のコードセルと同じようなコマンドを使い、括弧内を選んだテーマに書き換えるだけで変更可能です。
