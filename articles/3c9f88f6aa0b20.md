# 3DCGアルファベットアニメ制作をスクリプト化するまでの道のり
目標: 3DCGアルファベット**アニメーション**を作ること

## Abstract
- Blernder5.0で、[3Dモデル(文字"T")生成編](https://zenn.dev/rg687076/articles/1e79cd7474040f)で作成した[3Dモデル"T"](https://github.com/aaaa1597/Blender5AlphabetAmin.git)にリギングする手順

## 前提
- Blender5.0をインストール
- [ココ](https://github.com/aaaa1597/Blender5AlphabetAmin.git)から「char_T.blend」をDLしておく。

## 前準備
統計情報を表示しとく
![](https://storage.googleapis.com/zenn-user-upload/f323d7ed056f-20260105.png =500x)

## 1. アルファベット"T"のサイズ調整
このままボーンを追加していくと、Meshとボーンのサイズ感が合わないので、調整する。
![](https://storage.googleapis.com/zenn-user-upload/eed2a18cd775-20260110.png)
オブジェクト → 適用 → スケール で X,Y,Xのスケールを1.0にする。
![](https://storage.googleapis.com/zenn-user-upload/e03ba3c51ab8-20260110.png)

## 2. アルファベット"T"にボーン追加
### 2-0. 準備 ボーンが埋まって見えなくなるので最前面に設定
データタブ → 最前面にチェック
![](https://storage.googleapis.com/zenn-user-upload/c45dd1a9a772-20260110.png)

### 2-1. アーマチュア追加 → ボーン追加
アウトライナーで何も選ばない状態にする → 追加 → アーマチュア
※アーマチュアは、Mesh_Tとは全然関係ないオブジェクトなので何も選ぶ必要がない。(選んでても害はない)
![](https://storage.googleapis.com/zenn-user-upload/68f10f516911-20260105.png =500x)
アウトライナーでボーン名を設定
![](https://storage.googleapis.com/zenn-user-upload/e761d154545e-20260110.png =300x)

編集モード → テールのZを0.8mに設定
![](https://storage.googleapis.com/zenn-user-upload/fbf6502a9b00-20260110.png)

2つ目以降のボーン追加
編集モードのまま → テールの方を選択 → 追加 → 単一ボーン
![](https://storage.googleapis.com/zenn-user-upload/c00dd437e9ef-20260110.png)

親と接続を設定。変形も適宜設定。
![](https://storage.googleapis.com/zenn-user-upload/6e56724bbfa8-20260110.png)

同じ手順で下記のボーンを追加していく。
||ボーン名|部位|ヘッド|テール|親|接続|回転を継承|変形|
|---|---|---|---|---|---|---|---|---|
|1|Root|-|0,0,0|0,0,0.8|-|-|-|OFF|OFF|
|2|Waist|腰|0,0,8|0,0,1.2|Root|OFF|OFF|ON|
|3|Abs|腹|0,0,1.2|0,0,1.6|Waist|ON|ON|ON|
|4|Chest|胸|0,0,1.6|0,0,2|Abs|ON|ON|ON|
|5|LUpperArm|左上腕|0,0,2|0.4175,0,2|Chest|ON|ON|ON|
|6|LLowerArm|左下腕|0.4175,0,2|0.835,0,2|LUpperArm|ON|ON|ON|
|7|RUpperArm|右上腕|0,0,2|-0.4175,0,2|Chest|ON|ON|ON|
|8|RLowerArm|右下腕|0.4175,0,2|-0.835,0,2|RUpperArm|ON|ON|ON|
|9|UpperLeg|上腿|0,0,0.8|0,0,0.4|Waist|ON|ON|ON|
|10|LowerLeg|下腿|0,0,0.4|0,0,0|UpperLeg|ON|ON|ON|

### 2-2. ボーン回転伝播とロールの健全性確認
腰(Waist)と、左右上腕(LUpperArm/RUpperArm)、上腿(UpperLeg)をそれぞれ回転して下記を確認。
- ボーンの確認 ... 親子で一緒に回るか
- ロールの確認 ... 回転軸が直感通りか

ポーズモード → Waistを選択
![](https://storage.googleapis.com/zenn-user-upload/1f5577b51efb-20260111.png =500x)
Rキー押下 → Zキー押下 → マウスを左右に移動
![](https://storage.googleapis.com/zenn-user-upload/01dac8544c53-20260111.png =500x)
→ 腰(Waist)を回転させて、他のボーンも一緒に回転するかをチェックする。
※同様に、左右上腕(LUpperArm/RUpperArm)、上腿(UpperLeg)も回転させて変な動きしないかをチェックする。
今回は正しく動作するので問題なし!

## 3. Mesh とアーマチュアを関連付け(＝自動ウェイト付け)
### 3.0. 準備 ボーンの前面表示設定
ボーンは前面に見えてた方が何かと操作しやすいので設定しとく。
[何も選択する必要なく →] オブジェクト → 最前面
![](https://storage.googleapis.com/zenn-user-upload/2289d009b19d-20260117.png =500x)

### 3.1. Mesh とアーマチュアを関連付け(＝自動ウェイト付け)
オブジェクトモード → アウトライナーから、Armature→のMesh_Tの順で選択
![](https://storage.googleapis.com/zenn-user-upload/bc69240448a6-20260117.png =500x)
オブジェクト → ペアレント　→ 自動ウェイトで を実行
![](https://storage.googleapis.com/zenn-user-upload/a11e9d3c0c76-20260117.png =500x)
アーマチュアの配下にMesh_Tが移動したら成功!
※上手くいかない時は、選択順をMesh_T → アーマチュアの順に変更して自動ウェイトを実行すると成功したりする。
![](https://storage.googleapis.com/zenn-user-upload/6cfba7611c97-20260117.png =500x)

### 3.2. 自動ウェイトのウェイト確認
#### 準備 「オブジェクトのモードをロック」を解除
※これをしないとポーズモードのアーマチュアとMeshオブジェクトの同時選択ができない。
![](https://storage.googleapis.com/zenn-user-upload/cff715cdb400-20260117.png =300x)

アウトライナーからアーマチュアを選択 → ポーズモード → 3Dビューポート上のMesh_Tオブジェクトをクリック
![](https://storage.googleapis.com/zenn-user-upload/20034f8a49d3-20260117.png =500x)

クリックした後 → ウェイトペイントに変更
![](https://storage.googleapis.com/zenn-user-upload/c8be27810394-20260117.png =500x)

#### 確認
真っ黒になってるんだけども。。。orz
ここからウェイトペイントして調整していく
![](https://storage.googleapis.com/zenn-user-upload/fb017fd671cc-20260117.png =500x)

## 4. ウェイト調整(Weight Paint)
### 準備1 ウェイトの等高線を表示
これ表示させとくと、ウェイトが分かりやすくなる。
※ワイヤも表示させてみた
![](https://storage.googleapis.com/zenn-user-upload/e4118237ad93-20260117.png =400x)

### 準備2 自動正規化にチェック
アウトライナーからMesh_Tを選択 → ウェイトペイントモード → サイドバーのツール をクリック → オプション → 自動正規化
![](https://storage.googleapis.com/zenn-user-upload/8d79e391b504-20260117.png =500x)

### 準備3 対称にチェック
※お好みで...
サイドバーのツール をクリック → 対称 → Xをクリック
![](https://storage.googleapis.com/zenn-user-upload/901c58d56f66-20260117.png =500x)

### ■ウェイトペイントで修正していく
ウェイトを設定していくだけ。そもそも真っ黒(=何も設定されてない)なので気にせず設定していい。まずWaistから。
![](https://storage.googleapis.com/zenn-user-upload/f59afe4946dd-20260117.png =500x)
Waist(=腰部)あたりの頂点をクリックしていく。クリックしたところを中心に色がつくので確認しながら。
![](https://storage.googleapis.com/zenn-user-upload/cceeebaeeb42-20260117.png =500x)

### ■変形チェックもする。
いい感じにウェイト設定できたら、ボーンを動かして確認する。
Waist → ポーズモード → 回転。
![](https://storage.googleapis.com/zenn-user-upload/458438f0c732-20260117.png =500x)
適当な軸を回転させてMesh_Tの変形がおかしくないかをチェックする。
![](https://storage.googleapis.com/zenn-user-upload/8e6e67f6fc5c-20260117.png =500x)

Abs,Chest等他のボーンもウェイトを設定していく。(Rootは設定しない)

### ■ボーンの位置も確認する。
今回、変形チェックで分かったんだけど、腕のボーン([L|R]UpperArm/[L|R]LowerArm)って「T」横棒の厚みの真ん中に埋める必要があったのだけど、上面に張り付いて配置してた。
なので位置移動する。

Mesh_Tを選択 → 編集モード → 横棒の「上面」と「下面」の面を1枚ずつ選択(Shiftで2面選択)
![](https://storage.googleapis.com/zenn-user-upload/1f4ddbc9e91e-20260117.png =500x)
![](https://storage.googleapis.com/zenn-user-upload/1d3bba681284-20260117.png =500x)

上メニューのメッシュ → スナップ → "カーソル → 選択物"
![](https://storage.googleapis.com/zenn-user-upload/edb66c2a7d3a-20260117.png =500x)

3Dカーソルが横棒の厚みのちょうど中央にくる
![](https://storage.googleapis.com/zenn-user-upload/028d12e4b851-20260117.png =500x)

アーマチュア選択 → 編集モード → 移動したい横棒ボーンのヘッド or テールを選択 Shift + S → 選択物 → カーソル
![](https://storage.googleapis.com/zenn-user-upload/34d76da84498-20260117.png =500x)

選択したボーンのヘッド or テールが 3Dカーソルの位置に移動する
![](https://storage.googleapis.com/zenn-user-upload/2305fffda972-20260117.png =500x)

移動したいのはZ軸だけなので、X軸とY軸の値は0mに戻す。
![](https://storage.googleapis.com/zenn-user-upload/8fa36d3220c4-20260117.png =500x)

ほかのテールもZ軸の値(1.87097m)に変更する。
![](https://storage.googleapis.com/zenn-user-upload/086fb5fae299-20260117.png =500x)

腕ボーンの移動完了
![](https://storage.googleapis.com/zenn-user-upload/da9fd9d292f8-20260117.png)

変形チェックも確認して問題なければ完了。
