# Abstract
レイヤーサイズの変更から領域拡大する方法は簡単だったけど、拡大した部分が見えず戸惑ったので、残しときます。


# 手順
画像読込み
↓
[レイヤーパネル] → [レイヤサイズ変更]
↓
[メニュー] → [画像] → [キャンパスをレイヤーに合わせる]

:::message
↑これがポイント。これをしないと拡大領域が見えないままとなる。
:::
