# Abstract
Stable Diffusion forgeすごいいい。ControlNetが最初からついてきてるし。
だけど、img2imgで、ControlNetのinpaintとかの入力に画像が必要な機能に、画像設定する欄がなかったので探すのに苦しんで...
ので、ここに残します。

# 前提
- Stable Diffusion forgeのインストール
- Stable Diffusion forgeを簡単に使えること

# 結論
ControlNet Integratedを開いて、"Upload independent control image"をチェックする。

# 手順
img2img → ControlNet Integrated → ControlNet Unit 0をEnableに → "Upload independent control image"をチェック
![](https://storage.googleapis.com/zenn-user-upload/7ccb67969dc9-20240520.png)
↓
すると、入力欄が表示される。
![](https://storage.googleapis.com/zenn-user-upload/179bfc9b2fe1-20240520.png)
これで入力できるようになった。
