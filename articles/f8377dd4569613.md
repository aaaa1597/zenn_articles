# 手順
Reactに"npm i react-zxing"をインストール。
```shell:
$ cd ~ && mkdir react-qrcode && cd react-qrcode
$ npx create-react-app ./ --template typescript
$ npm init -y
$ npm i react-zxing
### $ npm start ### ←とりあえず動くのを見たかっただけ。
```
