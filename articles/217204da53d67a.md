# Abstract
無料でStable Diffusionの動かし方まとめ。
せっかくRTX4060のPCを買ったのに、Stable Diffusionを実行すると10分もたたずに黒画面になる。熱暴走が原因ってのがわかったけど、水冷化で対策できそうだけど、結構お金かかる。
ってことで、Stable Diffusionを無料で動かしてみた。

# 前提
- 電話番号が必要。アカウント作る時に求めてくる。
- Paperspace Gradientってサイトを利用するのでアクセスできる環境で。

# ざっくり手順
1. アカウントの作成
2. Notebookの作成
3. Forgeの環境を構築
4. 実行

# 詳細手順

[www.paperspace.com](https://www.paperspace.com/)にアクセス。
Googleアカウント持ってるので、それでログインする。

## 3. Forgeの環境を構築
webui2.ipynbをダウンロード
[paperspace-template/webui2.ipynb](https://github.com/javacommons/paperspace-template/blob/main/webui2.ipynb)にアクセス → ダウンロード
![](https://storage.googleapis.com/zenn-user-upload/5af47ec2d539-20240403.png =400x)
↓
webui2.ipynbをPaperspace Gradientにアップロード
![](https://storage.googleapis.com/zenn-user-upload/47c62d72692b-20240403.png =400x)
↓
アップロード完了
![](https://storage.googleapis.com/zenn-user-upload/0e0396968c0b-20240403.png =400x)
↓
ダブルクリックで開く。
![](https://storage.googleapis.com/zenn-user-upload/ac9cb6b4f76e-20240403.png =400x)
↓
git cloneのURLを変更する。
変更前 : !git clone https://github.com/AUTOMATIC1111/stable-diffusion-webui
変更後 : !git clone https://github.com/lllyasviel/stable-diffusion-webui-forge.git
![](https://storage.googleapis.com/zenn-user-upload/d31fe2b818b2-20240403.png =400x)
↓
RUN実行。なんかいろいろ出る。
![](https://storage.googleapis.com/zenn-user-upload/d3ec82b51b9c-20240403.png =400x)
↓
フォルダが出来てる。
![](https://storage.googleapis.com/zenn-user-upload/a09919752f87-20240403.png =400x)
↓
モデルのダウンロードのところのフォルダパスを修正。
変更前 : /notebooks/stable-diffusion-webui
変更後 : /notebooks/stable-diffusion-webui-forge
![](https://storage.googleapis.com/zenn-user-upload/cd5388d894c2-20240403.png)
↓
モデルを自分のモデルに変更。そのままだと4つもダウンロードするし、無料枠の5GBを超えてしまうので、なるだけ小さくて、自分好みの絵がでるモデルに変更する。
変更前 : !wget -nc https://civitai.com/api/download/models/16859 -O BlueberryMix-1.0.safetensors
変更後 : !wget -nc https://civitai.com/api/download/models/334282 -O kawaiiRealisticAsian_v07.safetensors
Run押下でモデルのダウンロードが始まる。
![](https://storage.googleapis.com/zenn-user-upload/8660aa32c7b4-20240403.png =400x)
↓
ダウンロードが完了したら、WebUI起動
![](https://storage.googleapis.com/zenn-user-upload/142978c13b30-20240403.png =400x)
↓
待つとWebUI起動する。Running on public URLのURLをクリックすると、使えるようになる。
![](https://storage.googleapis.com/zenn-user-upload/d53eb9aeb633-20240403.png =400x)
