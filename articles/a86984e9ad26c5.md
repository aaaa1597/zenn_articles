# データ型と欠損値
頻出する問題(欠損値)に対処します

## 理論編

### はじめに
このチュートリアルでは、DataFrame または Series 内のデータ型を調べる方法を学びます。また、エントリの検索と置換の方法も学びます。
**準備**
```python
import pandas as pd
reviews = pd.read_csv("../input/wine-reviews/winemag-data-130k-v2.csv", index_col=0)
pd.set_option('display.max_rows', 5)
```


#### データ型
DataFrame または Series 内の列のデータ型は、dtype と呼ばれます。
dtypeプロパティを使って特定の列の型を取得できます。例えば、reviewsデータフレームのprice列のdtypeを取得できます。
```python
reviews.price.dtype
```
**結果**
dtype('float64')

さらに、dtypes プロパティは、DataFrame 内のすべての列の dtype を返します。
```python
reviews.dtypes
```
**結果**
```
country        object
description    object
                ...  
variety        object
winery         object
Length: 13, dtype: object
```

データ型は、pandas がデータを内部的にどのように保存しているかを示します。float64 は 64 ビット浮動小数点数を使用していることを意味し、int64 は同様のサイズの整数を使用していることを意味します。

考慮しとくべき、 1つの特殊な性格があり、完全に文字列で構成される列には独自の型が割り当てられず、代わりにオブジェクト型が割り当てられることです。

astype() 関数を使用することで、列の型を別の型に変換することが可能です。変換が可能な場合は、いつでも変換可能です。例えば、points 列を既存の int64 データ型から float64 データ型に変換できます。
```python
reviews.points.astype('float64')
```
**結果**
```
0         87.0
1         87.0
          ... 
129969    90.0
129970    90.0
Name: points, Length: 129971, dtype: float64
```
DataFrame または Series インデックスにも独自の dtype があります。
```python
reviews.index.dtype
```
**結果**
```
dtype('int64')
```
Pandasは、カテゴリデータや時系列データといった、より特殊なデータ型もサポートしています。これらのデータ型はあまり使用されないため、このチュートリアルの後のセクションでは省略します。

#### 欠損値
欠損値のあるエントリには、NaN（Not a Number）という値が設定されます。技術的な理由により、これらのNaN値は常にfloat64型となります。

Pandasは欠損値に特化したメソッドをいくつか提供しています。NaNエントリを選択するには、pd.isnull()（または対応するpd.notnull()）を使用します。これは以下のように使用することを目的としています。
```python
reviews[pd.isnull(reviews.country)]
```
**結果**

欠損値の置き換えは一般的な操作です。Pandas はこの問題に対処するための非常に便利なメソッド、fillna() を提供しています。fillna() は、このようなデータへの影響を軽減するためのいくつかの異なる戦略を提供しています。例えば、NaN を「Unknown」に置き換えるだけで済みます。
```python
reviews.region_2.fillna("Unknown")
```
**結果**
0         Unknown
1         Unknown
           ...   
129969    Unknown
129970    Unknown
Name: region_2, Length: 129971, dtype: object

あるいは、データベース内の特定のレコードの後に​​出現する最初の非NULL値で、各欠損値を埋めることもできます。これはバックフィル戦略と呼ばれます。

さらに、置き換えたい非null値があるかもしれません。例えば、このデータセットが公開されて以降、査読者のKerin O'KeefeがTwitterのハンドルを@kerinokeefeから@kerinoに変更したとします。これをデータセットに反映する 方法の1つは、replace() メソッドを使用することです。
```python
reviews.taster_twitter_handle.replace("@kerinokeefe", "@kerino")
```
**結果**
0         @kerino
1         @vossroger
             ...    
129969    @vossroger
129970    @vossroger
Name: taster_twitter_handle, Length: 129971, dtype: object

replace() メソッドは、データセット内で「不明」、「未公開」、「無効」など、何らかのセンチネル値が与えられた欠落データを置き換えるのに便利です。
