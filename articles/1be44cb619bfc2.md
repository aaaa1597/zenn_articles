## 名前の変更と結合
データはソースが様々です。それらをまとめて理解できるようになりましょう。

### 理論編
#### はじめに
多くの場合、データは列名、インデックス名、その他の命名規則に従って提供され、それらに満足できないことがあります。その場合は、pandas 関数を使用して、問題のあるエントリの名前を適切なものに変更する方法を学習します。
また、複数の DataFrame や Series からのデータを結合する方法についても説明します。

**準備**
先に下記コードを実行しておきます。
```python
import pandas as pd
pd.set_option('display.max_rows', 5)
reviews = pd.read_csv("../input/wine-reviews/winemag-data-130k-v2.csv", index_col=0)
```

#### 名前の変更(rename())

```python
reviews.rename(columns={'points': 'score'})
```

#### 結合(Combining)
データセットに対して操作を実行する際、異なる DataFrame や Series を複雑な方法で結合する必要がある場合があります。Pandasには、これを行うための3つのコアメソッドがあります。複雑さが増す順に、concat()、join()、merge()です。join() は簡単に実行できるため説明を省略し、残り 2 つの関数に焦点を当てます。

##### concat()
最も単純な結合方法はconcat()です。要素のリストが与えられると、この関数はそれらの要素を軸に沿って結合します。これは、異なるDataFrameまたはSeriesオブジェクトに同じフィールド（列）を持つデータがある場合に便利です。
例えば、YouTube 動画データセットが挙げられます。このデータセットでは、データが原産国 (この例ではカナダと英国) に基づいて分割されています。
複数の国を同時に調査したい場合は、concat() を使用してそれらをまとめることができます。
```python
canadian_youtube = pd.read_csv("../input/youtube-new/CAvideos.csv")
british_youtube = pd.read_csv("../input/youtube-new/GBvideos.csv")

pd.concat([canadian_youtube, british_youtube])
```

##### join()
次に複雑な結合子は join() です。
join() を使用すると、共通のインデックスを持つ異なる DataFrame オブジェクトを組み合わせることができます。たとえば、カナダとイギリスの両方で同じ日にトレンドになった動画を取得するには、次のようにします。

```python
left = canadian_youtube.set_index(['title', 'trending_date'])
right = british_youtube.set_index(['title', 'trending_date'])

left.join(right, lsuffix='_CAN', rsuffix='_UK')
```
