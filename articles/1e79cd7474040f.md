# 3DCGアルファベットアニメ制作をスクリプト化するまでの道のり
目標: 3DCGアルファベット**アニメーション**を作ること

## Abstract
- Blernder5.0で、アルファベット"T"のMESH作成手順

## 背景
Pixerのロゴアニメみたいな、アルファベットアニメをあまり手をかけずにサクッと作れるようになりたい。最終的にはBlenderのpythonスクリプトで作るつもりなんだけど、Blenderの操作方法が分かんないと、どうにも捗らない。
まずは、アルファベット"T"の3DCGモデルを生成しようと思うのだけど、皮のみ3DCGモデルを生成するにしても、アニメーションを意識してポリゴン分割する必要があるっての全く知らんかった。アルファベット"T"だから最小構成ポリゴンでつくればいいやんって思ってたけど、そうじゃなかった。

## 前提
- Blender5.0をインストール

## ①アルファベット"T"のMESH作成
### ①-1. テキストオブジェクト"T"を生成
アルファベットはなんでもいいのだけど、とりあえず"T"にしてみた
#### ①-1.1. 一旦削除
Blender5起動 → アウトライナーでCubeを選択 → Delキーで削除
※CameraとLightは有っても無くてもどっちでもいいので残しておく。
![](https://storage.googleapis.com/zenn-user-upload/cab4f7dc31b3-20251231.png)

#### ①-1.2. テキストオブジェクト追加
3Dビューポート上部メニューの追加 → テキストを選択
![](https://storage.googleapis.com/zenn-user-upload/d4d85bcbe00d-20251231.png)

#### ①-1.3. 文字列を"T"に変更
アウトライナーで"テキスト"を選択 → 編集モードに変更
![](https://storage.googleapis.com/zenn-user-upload/a1d6dde493c6-20251231.png)
キーボードからBackSpaceで文字列削除 → 「T」を入力 → オブジェクトモードに戻す
![](https://storage.googleapis.com/zenn-user-upload/7c415e28bf6f-20251231.png)

### ①-2. 厚みをつける(立体化)
プロパティパネル → オブジェクトデータプロパティ → ジオメトリを開く → 押し出し0.1mを設定
![](https://storage.googleapis.com/zenn-user-upload/95117ef1512b-20251231.png =400x)

### ①-3. メッシュに変換
引続きアウトライナーで"テキスト"を選択しとく → オブジェクト → 変換 → メッシュ
![](https://storage.googleapis.com/zenn-user-upload/34c55b23360c-20251231.png =500x)

### ①-4. オブジェクト名変更
作成したMeshが、アウトライナーでは"テキスト"と"テキスト.001"になっているので適当な名称に変更する。
![](https://storage.googleapis.com/zenn-user-upload/604ad89abead-20251231.png)

アルファベット"T"のMESH作成完了。

## ②多ポリゴン化
アニメーションと一言でいっても方向性があるみたいで。
- "ぐにゃっ"と変形する擬人化アニメを目指すなら、ポリゴン分割(Subdivide) → 再メッシュ(Remesh)がいい。
- 「T」の手足がばらばらに動くの目指すなら Mesh をパーツごとに分割した方がいい。

ってCopilot君が言ってた。

## ③原点調整/姿勢起こし
### ③-1. X位置移動
### ③-1.1. X位置を中心に移動
3Dビューポートの何もないところを右クリック → 原点を設定 → ジオメトリを原点へ移動
※Y軸にも移動してしまうけど、それは次の工程で修正する。
![](https://storage.googleapis.com/zenn-user-upload/1028541a2680-20260103.png =400x)

### ③-1.2.「T」の下端をY=0の位置に移動
#### ③-1.2.1. オブジェクトモード → アウトライナーのMesh_Tを選択
![](https://storage.googleapis.com/zenn-user-upload/43bf2f92344c-20260103.png =500x)

#### ③-1.2.2. 編集モード → 下端に「基準点」を置く
##### ③-1.2.2.1 選択モード:面を選択 → 下端の面を全て選択
![](https://storage.googleapis.com/zenn-user-upload/75f29faf5f89-20260103.png =400x)
##### ③-1.2.2.2 メッシュ → スナップ → カーソル → 選択物
![](https://storage.googleapis.com/zenn-user-upload/235ec0e467aa-20260103.png =400x)
##### 3Dカーソルが、下端の中央に移動
※ まだメッシュは動かない
![](https://storage.googleapis.com/zenn-user-upload/eacd92235869-20260103.png =200x)

#### ③-1.2.3. オブジェクトモードに戻る
![](https://storage.googleapis.com/zenn-user-upload/173206920166-20260103.png =400x)

#### ③-1.2.4. 原点を3Dカーソルに移動
3Dビューポートの何もないところを右クリック → 原点を設定 → 原点を3Dカーソルへ移動
![](https://storage.googleapis.com/zenn-user-upload/4bbe595d666d-20260103.png =500x)
✔ 原点が「下端中央」に来る
※見た目は変わらない

#### ③-1-2.5. オブジェクト全体をY=0へ移動
Nパネル → Transform → 位置 → Y=0を設定
![](https://storage.googleapis.com/zenn-user-upload/57b933006b69-20260103.png =500x)
移動完了!

### ③-1-3. 3Dカーソルを原点に戻す
オブジェクト → スナップ → "カーソル → ワールド原点"
![](https://storage.googleapis.com/zenn-user-upload/0b7ae93a5ac2-20260103.png =500x)

### ③-2. X軸90°回転で起こす
#### ③-2.1. 見た目をX軸90°回転で起こす
オブジェクト → Mesh_T → Nキー押下(サイドパネル表示) → X軸を90°
![](https://storage.googleapis.com/zenn-user-upload/d60c10b6ac5d-20260105.png =500x)

#### ③-2.2. その姿勢を「基準」にする
Mesh_T → オブジェクト → 適用 → 全トランスフォーム
![](https://storage.googleapis.com/zenn-user-upload/256ed51ffaee-20260105.png =500x)

![](https://storage.googleapis.com/zenn-user-upload/daaef6d61201-20260105.png =500x)
トランスフォームの位置,回転,スケールが全部初期値になって、見た目の位置は期待の位置になった。
