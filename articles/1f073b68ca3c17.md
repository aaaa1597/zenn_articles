# Abstract
「Copilot」の意味をまとめてみた。とは言っても副操縦士の意味でなくて、技術ネタとしての「Copilot」のこと。
先日「Copilotがすごい」って、話しかけられたんだけど、何のことがさっぱり分からず...。
こういう時、その場で「そうなんですか、へー」って流して、場面が変わった時にシレって調べるんだけど、ホントに分かんなかった。
なので、「Copilot」まわりの言葉たちをまとめてみた。

# 結論
Microsoftの製品群に追加された「Copilot」ってAI機能のこと。

# なぜ分からないのか？
それもそのはずで、Copilotで検索すると、windowsのCopilotやら、githubのCopilotやらホントいろいろ出てくる。一体どれのこと？「イー」ってなる。

## 何のことない
結局**全部**、Microsoftの製品が絡んでいるという。githubもMicrosoftってのは知らんかった。(お世話になってます。)

ざっくりとはこんな使い方。(もちろん例外はある、Edge in copilotとか)
![](https://storage.googleapis.com/zenn-user-upload/bd1da5399d86-20240323.png)
copilotの操作画面で、"表のまとめ方教えて"とか質問打ち込むと、返事が返ってくるサービス。絵を描いてって言ったら、描いてくれたりもする。便利。

# まとめ
「Copilot」は、MicrosoftのAIサービス。
会話の中で出てきたら、文脈から判断する。例えばExcelの話してて「Copilot」がでてきたら"Excel in Copilot"だし、Edgeの話をしてて「Copilot」がでてきたら"Edge in Copilot"。
いきなり、「Copilotがすごい」みたいに話す人がいたら、次の言葉を待とう。いろいろありすぎて何のことを言っているか、分からないから。(たぶん本人も分かってない...)
