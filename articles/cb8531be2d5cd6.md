# Abstract
React+TypeScriptのプロジェクトの最初って、create-react-appやって、EsLintインストールして、prettierインストールしてって、お決まりのパターンになってて。
だったらgithubに雛形登録しとって、そっから始めた方が早いやん。って事で。

# 前提
- gitインストール
- React環境構築済 [[環境構築]Ubuntu22.04+VSCode+React+TypeScriptの開発環境を構築してみた。](https://zenn.dev/rg687076/articles/5912e808924d28)

# githubリポジトリ
https://github.com/aaaa1597/React-Ts-Template.git

# 始め方
プロジェクト名、aaaaa。
```shell:
$ cd ~
$ git clone https://github.com/aaaa1597/React-Ts-Template.git
$ mv React-Ts-Template aaaa
$ cd aaaa
$ npm install
$ npm start
```
あとはVSCodeで開発するなり適当に。

# 中身は↓だから。
1. プロジェクト生成(npx create-react-app ./ --template typescript)
2. EsLintインストール
3. prettierインストール
4. eslint-config-prettierインストール
5. eslint-config-prettier の設定ファイルを修正
6. prettierの設定ファイル".prettierrc"を手で作成
7. VSCodeもlaunch.jsonを生成
8. prop-typesインストール
9. ソースコード削除しまくってキレイに。
