# 手順
## モジュール更新とか
```shell:ubuntu初期化のおまじない
#パッケージ更新、自動削除、キャッシュクリーン
$ sudo apt update && sudo apt full-upgrade -y && sudo apt autoremove -y && sudo apt autoclean -y
#残留ファイルをすべて削除
$ sudo apt purge -y $(dpkg -l | grep '^rc' | awk '{print $2}')
```

## 必要なパッケージのインストール
```shell:wget gpgをインストール
$ sudo apt -y install wget gpg apt-transport-https
```
## 署名鍵の登録
```shell:
$ wget -qO- https://packages.microsoft.com/keys/microsoft.asc | gpg --dearmor > packages.microsoft.gpg
$ sudo install -D -o root -g root -m 644 packages.microsoft.gpg /etc/apt/keyrings/packages.microsoft.gpg
$ rm -f packages.microsoft.gpg
```
## リポジトリの登録
```shell:リポジトリ登録
$ sudo sh -c 'echo "deb [arch=amd64,arm64,armhf signed-by=/etc/apt/keyrings/packages.microsoft.gpg] https://packages.microsoft.com/repos/code stable main" > /etc/apt/sources.list.d/vscode.list'
$ cat /etc/apt/sources.list.d/vscode.list
deb [arch=amd64,arm64,armhf signed-by=/etc/apt/keyrings/packages.microsoft.gpg] https://packages.microsoft.com/repos/code stable main
```

## vscodeインストール
```shell
$ sudo apt update
$ sudo apt -y install code
```
## vscode起動
```shell
$ code
```
