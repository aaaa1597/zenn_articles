# Abstract
アイビスペイントの初めの一歩。　起動から最初の1ページにたどり着くまで。

# 経緯
Stable Diffusionを使えるようになって、マンガに挑戦してみようかと思い立った今日この頃。
マンガ描くならツールはクリスタがいいらしいのだけど、高くって高くって。無料でも使えるアイビスペイントで始めてみた。
けど、起動しても何すりゃいいのかさっぱりわからん。なので最初の1ページを描くまでの手順を残しときます。

# 手順

　インストールは迷わないから。適当に。
