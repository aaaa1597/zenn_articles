### ヒストグラム
例えば、アヤメの花びらの長さがどのように変化するかを見るためのヒストグラムを作成したいとします。これは sns.histplot コマンドで実行できます。

```python
# Histogram 
sns.histplot(iris_data['Petal Length (cm)'])
```
![](https://storage.googleapis.com/zenn-user-upload/43a7efd0cf4d-20260204.png)

上記のコード セルでは、プロットする列をコマンドに指定する必要がありました (この場合は、「花びらの長さ (cm)」を選択しました)。

### 密度プロット（Density plots）
次にご紹介するグラフの種類は、カーネル密度推定（KDE）プロットです。KDEプロットに馴染みがない場合は、「滑らかになったヒストグラム」と考えてみてください。

KDEプロットを作成するには、sns.kdeplot コマンドを使用します。shade=True（※現在のバージョンでは fill=True が推奨）を設定すると曲線の下の領域が塗りつぶされ、data= でプロットしたい列を選択します。

```python
# KDE plot 
sns.kdeplot(data=iris_data['Petal Length (cm)'], shade=True)
```

### 2次元KDEプロット
KDEプロットを作成する際、対象は単一の列（変数）に限定されません。sns.jointplot コマンドを使えば、2次元（2D）のKDEプロットを作成することができます。

下のグラフでは、色分けによって「がく片の幅（sepal width）」と「花弁の長さ（petal length）」のさまざまな組み合わせが、どれくらいの確率で出現するかを示しています。図の色の濃い部分ほど、その組み合わせが出現する可能性が高いことを表しています。

```python
# 2D KDE plot
sns.jointplot(x=iris_data['Petal Length (cm)'], y=iris_data['Sepal Width (cm)'], kind="kde")
```

![](https://storage.googleapis.com/zenn-user-upload/8f1a2048e5b3-20260204.png)

中央の「2次元KDEプロット」に加えて、以下の点にも注目してください。

図の上部にある曲線は、x軸のデータ（この場合は iris_data['Petal Length (cm)']）のKDEプロットです。

図の右側にある曲線は、y軸のデータ（この場合は iris_data['Sepal Width (cm)']）のKDEプロットです。

### 色分けされたプロット
チュートリアルの次のパートでは、アヤメの品種（species）間の違いを理解するためのプロットを作成します。 sns.histplot コマンドを使用することで、花弁の長さ（petal length）に関する3つの異なるヒストグラム（各品種につき1つ）を作成できます。

- data=: データの読み込みに使用した変数名を指定します。
- x=: プロットしたいデータが含まれる列の名前を指定します。
- hue=: データを異なるヒストグラムに分割（色分け）するための列を指定します。

```python
# Histograms for each species
sns.histplot(data=iris_data, x='Petal Length (cm)', hue='Species')

# Add title
plt.title("Histogram of Petal Lengths, by Species")
```

![](https://storage.googleapis.com/zenn-user-upload/cc6d471ee88a-20260204.png)

sns.kdeplot を使えば、上記のように各種のKDEプロットを作成することもできます。データ、x、hue の機能は、上記の sns.histplot を使用した場合と同じです。さらに、shade=True を設定することで、各曲線の下の領域に色を付けることができます。

```python
# KDE plots for each species
sns.kdeplot(data=iris_data, x='Petal Length (cm)', hue='Species', shade=True)

# Add title
plt.title("Distribution of Petal Lengths, by Species")
```

![](https://storage.googleapis.com/zenn-user-upload/a69e442aab9f-20260204.png)

グラフから見て取れる興味深いパターンの一つは、植物が2つのグループのいずれかに属しているように見える点です。**Iris versicolor（ブルーフラッグ）** と **Iris virginica（バージニカ）** は花弁の長さ（petal length）が似たような値を示していますが、**Iris setosa（ヒオウギアヤメ）** だけは完全に別のグループに属しています。

実際、このデータセットに基づけば、花弁の長さを見るだけで、あるアヤメが（versicolor や virginica ではなく）Iris setosa であると分類することさえ可能かもしれません。もしアヤメの花弁の長さが 2 cm 未満であれば、それはほぼ間違いなく Iris setosa であると言えるのです！
